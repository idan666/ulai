Ulai is 'Maybe' in Hebrew,
Randomly returns boolean response, 'true' or 'false', in javascript. No Errors and no tests Required!

## Install

```
$ npm install @idan666/ulai
```

## Usage

```js
const ulai = require("@idan666/ulai");

ulai();
//=> true
//=> false
